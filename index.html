jQuery Inline Currency (JIC) Demo

First of all, move each price into span element:

<span class="jic">$10</span>

And then, apply inlineCurrency plugin to your elements:

$(document).ready(function () { $('.jic').inlineCurrency(); });

Result:

You can buy it for $10

You can use inline parameters for special cases:

<span class="jic" data-jic-value="10" data-jic-currency="EUR" data-jic-convert-to="rub, usd, chf">Ten euros</span>

Result:

Ten euros

If you use decimal or thousands separators in prices, you can set config:

$(document).ready(function () { $('.jic-test').inlineCurrency({ "thousandsSplit": " ", "decimalsSplit": "," }); });

Result:

$100 000,20
